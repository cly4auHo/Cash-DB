1)Создать экземпляр класса:
cmd в папке src, javac папкаПроекта/пакет/ИмяКласса.java
Запустить класс (с мейном)
cmd в папке src, java папкаПроекта.пакет.ИмяКласса

2)Создать jar
cmd в папке src, jar cf имя джарника класс.class
тамже,jar cfm имяДжарника имяМанифеста пакет/класс.class
тамже, jar tf имя джарника
тамже, jar xf имя джарника
тамже, java -cp имя джарника имя пакета.ИмяКласса
